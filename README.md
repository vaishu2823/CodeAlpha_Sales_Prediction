# 📈 Sales Prediction using Machine Learning

## 📌 Overview
This project predicts product sales based on advertising expenditures across multiple platforms such as TV, Radio, and Newspaper.

The project demonstrates how Machine Learning can help businesses analyze marketing effectiveness and forecast future sales.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Features
✅ Data Cleaning  
✅ Sales Distribution Analysis  
✅ Correlation Heatmap  
✅ Linear Regression Model  
✅ Feature Importance Analysis  
✅ Interactive Sales Prediction  
✅ Business Insights
